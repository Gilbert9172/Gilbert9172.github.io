## 와인 에티켓

- 와인 받을 때, 잔을 들지 않고, 잔의 받침 부분에 가볍게 손을 올린다.
- 다 따랐을 때 가벼운 목례나 감사 인사 정도로 충분
- 1/3만 따른다.
- 첨잔을 한다. ( 따라준다고 원샷하지 마세요. )
- 테이스팅과 드링킹을 혼동하지 않는다
