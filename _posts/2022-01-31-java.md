### 🔎 ***for 문***

> for (초기화식; 조건식; 중감식)

초기화식, 조건식, 증감식이 두 개일 경우 , 로 구분한다.

```java
int sum = 0;
for (int i = 1; i<=100; i++) {
    sum = sum + i;
}
System.out.println(sum);
```

<br>

- 중첩 for 문

```java
for (int i = 1; i < 10; i++) {
    for (int k = 2; k <10, ++) {
        System.out.println(i+j);
    }
}
```
